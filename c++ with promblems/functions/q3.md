Simple Calculator Using Functions
Create four separate functions: add, subtract, multiply, and divide — each taking two double values. In main(), ask the user to enter two numbers and an operator (+, -, *, /), then call the matching function and display the result. Handle division by zero.

Sample Input
Enter num1: 8
Enter num2: 4
Operator: /
Expected Output
Result: 2
💡
Hint: Use a char variable for the operator. Use if-else if or a switch statement in main() to decide which function to call. In divide(), check if the second number is 0 before dividing.
