Greet by Name
Write a void function called greetUser that takes a string name as a parameter and prints a personalized greeting. Call it twice with different names.

Sample Input
greetUser("Riya")
greetUser("Aarav")
Expected Output
Hello, Riya! Welcome!
Hello, Aarav! Welcome!
💡
Hint: The function returns nothing (void), so no return needed. Include #include <string> at the top to use the string type.
