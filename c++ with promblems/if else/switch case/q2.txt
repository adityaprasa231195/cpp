Problem:
Write a basic calculator program. Take two integers and an operator (+, -, *, /) as input and display the result using a switch statement.
Special Rules:

If operator is / and second number is 0 → print "Math Error: Division by zero!"
If operator is invalid → print "Unknown operator!"

Sample Input/Output:
Input: 15, 3, /
Output: Result = 5

Input: 10, 0, /
Output: Math Error: Division by zero!

Input: 4, 5, ^
Output: Unknown operator!S
