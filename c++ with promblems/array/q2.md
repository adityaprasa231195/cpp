Count Even and Odd Numbers
Given an array of 6 integers, count how many numbers are even and how many are odd. Print both counts.

Input
arr = {3, 8, 5, 12, 7, 4}
Output
Even count: 3
Odd count: 3
Hint: Use two counter variables outside the loop. Inside the loop, use if (arr[i] % 2 == 0) to check even. Increment the correct counter each iteration.
