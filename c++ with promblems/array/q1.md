Print Array in Reverse Order
You are given an array of 5 integers. Using a for loop, print all the elements from the last to the first (in reverse order).

Input
arr = {10, 20, 30, 40, 50}
Output
50 40 30 20 10
Hint: Instead of starting i from 0, what if you start i from size-1 and decrease it each time? What should your condition be?
