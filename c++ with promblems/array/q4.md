Replace Negative Numbers with Zero
Given an array of 6 integers (which may contain negative numbers), replace every negative number with 0 and then print the modified array.

Input
arr = {3, -5, 8, -1, 0, -9}
Output
3 0 8 0 0 0
Hint: Inside the loop, check if arr[i] < 0. If yes, assign arr[i] = 0. You are modifying the array itself during iteration — this is completely valid!
