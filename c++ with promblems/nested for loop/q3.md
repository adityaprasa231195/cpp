The Multiplication Table Generator

Problem Statement: Print the multiplication tables for numbers 1 through 3, up to times 3.

Sample Output:

Plaintext
1 x 1 = 1
1 x 2 = 2
1 x 3 = 3
2 x 1 = 2
... (up to 3 x 3)
Hint: Use i for the first number (1 to 3) and j for the second number (1 to 3). Inside the inner loop, calculate and print i * j.
