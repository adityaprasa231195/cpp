Problem:
Write a program that acts as a simple calculator. Take two numbers and an operator (+, -, *, /) as input. Perform the operation and display the result.
Special Rules:

If the operator is / and the second number is 0, print "Division by zero is not allowed!"
If an invalid operator is entered (anything other than +, -, *, /), print "Invalid operator!"

Sample Input/Output:
Input: 10, 2, *
Output: Result = 20

Input: 10, 0, /
Output: Division by zero is not allowed!

Input: 5, 3, ^
Output: Invalid operator!
