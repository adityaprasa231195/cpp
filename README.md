# cpp

building consistency and discipline

## About

This repository contains C++ projects and exercises focused on improving coding consistency, discipline, and craftsmanship. The code is intended to be simple, well-structured, and easy to build so you can focus on learning and iteration.

## Language

- C++ (100%)

## Features

- Small, focused C++ programs
- Clear build instructions
- Examples and exercises to build consistency in coding practices

## Getting started

Prerequisites:

- A C++ compiler (g++, clang++) supporting C++11 or later
- Make (optional)

Clone the repository:

```bash
git clone https://github.com/adityaprasa231195/cpp.git
cd cpp
```

## Build

If there's a simple single-file program, compile it with:

```bash
g++ -std=c++17 -O2 -Wall -Wextra -o program src/main.cpp
```

If the project provides a Makefile, build with:

```bash
make
```

## Running

After building, run the produced binary:

```bash
./program
```

Adjust the path/name as needed based on the project structure.
